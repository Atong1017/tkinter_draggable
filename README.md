使用tkinter時，常因排版浪費很多時間，所以創建一份拖曳選單的功能，先大致排版再細部微調，以達節省時效。

功能:

1、(Combobox, Entry, Label, Button, Checkbutton)可拖曳、調整大小

2、可預存、加載、編輯/刪除執行中的code

3、可將調整好的存為.py，即可直接編寫code
